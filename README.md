# Ejercicio Toma De Decisiones Basadas En Datos

En el entorno empresarial actual, la capacidad de transformar grandes volúmenes de datos en información accionable es una ventaja competitiva crítica. Este ejercicio se centra en el análisis de un conjunto de datos de 500 clientes, diseñado para simular variables demográficas, económicas y de comportamiento de compra en las principales ciudades de México. El objetivo primordial es identificar patrones que permitan al área comercial optimizar sus estrategias de venta y marketing.

### Objetivo del análisis
El presente estudio busca responder a interrogantes clave del negocio, tales como:

1. Identificación de Perfiles: Determinar si variables como la edad, el género o el nivel de ingresos influyen directamente en la probabilidad de conversión.
2. Rendimiento Geográfico: Evaluar el desempeño comercial en distintas plazas (CDMX, Monterrey, Guadalajara, Puebla y Tijuana) para priorizar esfuerzos regionales.
3. Correlación de Factores: Analizar la relación estadística entre la frecuencia de visitas y el ticket promedio de gasto para entender la lealtad del cliente.
